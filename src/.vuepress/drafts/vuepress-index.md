# Vuepress简介

Vuepress是由Vue驱动的静态页面生成工具。[Evan][evan]最初是作为个人项目进行开发，目的是用来快速生成Vuejs子项目的文档。项目发布之后反响很大，一度进入[Github Trending][trending]，于是就决定将[Vuepress][vuepress]作为Vuejs的子项目之一了。
